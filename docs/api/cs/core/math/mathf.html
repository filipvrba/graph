Mathf

Pomocná třída pro výpočet úhlů, procenta a interpolace.

Popis

Mathf je pomocná tříta pro výpočet desetiných čísel. Zde najdeme pomocné funkce pro výpočet úhlů, procenta a interpolace. Tyto funkce jsou statické a lze u nich přistupovat přímo.

Příklad

const percentage = Mathf.percentage( this.value, this.totalValue ); const degrees = Mathf.per2deg( percentage ); const endRadian = Mathf.radians( degrees + this.endDegrees );

Úhel se převede na radián.

Například 90deg = Mathf.radians(90).toFixed(3);rad.

Radián se převede na úhel.

Například Mathf.radians(90).toFixed(3);rad = const radian = Mathf.radians(90); Mathf.degrees(radian); deg.

Procento se převede do úhlu.

Například 45% = Mathf.per2deg( 45 );deg.
