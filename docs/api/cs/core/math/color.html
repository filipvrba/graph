Color

Třída pro náhodilost barev.

Popis

Barvy jsou automaticky generovány, nebo jsou konstantně napsány, tak aby odpovídaly paletě barev. Třda nám umožní generovat náhodilost barev, která se ukládá do mezipaměti a nebude se opakovat, při budoucím volání.

Příklad

// Vytváříme novou třídu a potom dostáváme náhodnou barvu z palety barev. const color = new Color(); const randColorPalet = color.getRandomPalettte(); // Dostaneme náhodně vygenerovanou barvu. const randColor = Color.ranColorStyle();
výchozí:

Paleta zughy má délku const color = new Color(); color.colors.length; hex barev, a jsou uložené do řetězce.

výchozí:0

Ukládá unikátní bit pro identifikaci použitých barev z palety.

Konstruktor umožní ukládat už použité barvy z palety pomocí bitMask.

Náhodně vybere barvu z palety a přiřadímu unikátní bit, pro uložení do bitMask vlastnosti. Použitá barva v budoucím volání se neaplikuje.

Počet volání je omezený na const color = new Color(); color.colors.length; , z důvodu maximální velikosti palety barev. Jinak tato metoda vyhodí podmínku chyby.

Vrací string ve formátu hex.

Náhodně vygenerovaná barva ve formátu rgb.

Vrací string a dá se aplikovat na css barvu.
