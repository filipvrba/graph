Vector2

Vector se používá především v matematice.

Popis

Dvou elementární struktura která se používá k reprezentaci 2D pozice v prostoru, nebo reprezentaci jakéhokoliv jiných číselných hodnot ( např. výška a šířka ).

Popis Funkcí

  • Vector2 ( x, y )

Konstruktor vytvoří Vector2 se souřadnicemi x a y.

  • Vector2 add ( vector )

Připočtou se vektorové souřadnice a vrátí aktuální vektor.

  • Vector2 clone ()

Vytvoří novou třídu Vector2 a zkopíruje parametry x a y. Vrací nový Vector2.

  • distanceTo ( vector )

Vypočítá vzdálenost z aktuálního vektoru do definovaného vektoru.

  • distanceToSquared ( vector )

Vypočítá čtvercovou vzdálenost z aktuálního vektoru do definovaného vektoru. Tato metoda je rychleší pro výpočet než metoda distanceTo.


Použiteji pokud chcete porovnat vektory, nebo čtvercovou vzdálenost pro nějaký vzorec.

  • dot ( vector )

Vypočítá dot product z aktuální a definovaného vektoru. Tato metoda lze uplatnit k porovnávání úhlů mezi dvěma vektory.

  • equals ( x, y )

Porovná souřadnice x a y s aktuálním vektorem. Vrací true pokud, jsou souřadnice vektorů stejné.

  • length ()

Vrátí délku z aktuálního vektoru.

  • Vector2 lerp ( vector, alpha )

Lineárně interpoluje mezi vektorama, a to pomocí alpha což je procentuální vzdálenost.


Aktuální vektor interpoluje k definovanému vektoru. Hodnota alpha je interpolační hodnota která je normalizována. Pohybuje se v rozmezích 0 až 1.

  • Vector2 multiply ( vector )

Vynásobý aktuální vektor z definavaného vektoru.

  • Vector2 multiplyScalar ( scalar )

Vynásobý aktuální vektor skalárem.

  • Vector2 normalize ()

Aktuální vektor se převede na jednotkový vektor. Tento vektor bude mít stejný směr, ale s délkou 1.

  • Vector2 normalized ()

Tato metoda vytvoří nový Vector2 a vrátí normalizovaný vektor.

  • Vector2 sub ( vector )

Definovaný vektor odečte aktuální souřadnice x & y vektoru.

  • Vector2 subScalar ( scalar )

Definovaná hodnota skaláru odečte aktuální souřadnice x & y vektoru.
