Vector2

Vector se používá především v matematice.

Popis

Dvou elementární struktura která se používá k reprezentaci 2D pozice v prostoru, nebo reprezentaci jakéhokoliv jiných číselných hodnot ( např. výška a šířka ).

Příklad

// Získá veřejnou pozici z vlastní třídy a vytvoří nový konstruktor třídy Vector2. const positionExample = this.globalPosition.clone(); // Vytváříme novou třídu Vector2 a přidáváme parametry z pozice myši, která je volána posluchačem. const positionMouse = new Vector2( eventMouse.x, eventMouse.y ); // Zde vypíšeme do konzole vzdálenost mezi dvěma vektory. console.log( positionPiece.distanceTo( positionMouse ) );
