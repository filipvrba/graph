Detekce kolize.

Popis

Zpracovává detekci kolize, přes vstupní událost a je komponentou pro nový objekt.

Vysílá se, když je pozice myši vstoupena do areálu kolize.

Vysílá se, když je pozice myši odstoupena od areálu kolize.
výchozí:false

Pokud myš vstoupí do areálu kolize, tak vrací true.

Konstruktor dědí z tříd, inicializuje vlastnost a nemá žádné parametry.

vektorova pozice myši.

Virtuální funkce kontroluje přístupnost pozice myši, ve vyhrazeném areálu kolize pomocí logických výrazů.


Tyto výrazy se vyhodnocují podle inputPickable vlastnosti a isCollide ( ... ) funkcionality. Pokud výrazy jsou pravdivé, tak se spouští mouseEntered funkce. Pokud ne, tak se spouští mouseExited funkce

Spouští mouseEntered signál.

Spouští mouseExited signál.

vektorova pozice myši.

Kontroluje pozici myši, jestli je v areálu kolize.


Areál kolize je vypočítána dynamicky. Při každém spuštění této funkce, se inicializují konstantní hodnoty, pro výpočet vzdálenosti kořene areálu, směr myši (který se předává do atributu funkce) a radiánu (který je nakonec vyhodnocován).


vrací logický výraz z vyhodnocených radiánů.
