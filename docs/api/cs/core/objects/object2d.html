basicObject-dispatcher

2D objekt s transformací pozice.

Popis

Všechny 2D objekty, pomocí této třídy se mohou přesouvat na určité pozice v prostoru scény.

výchozí:Vector2( 0, 0 )

Lokální 2D pozice, která je vzhledem k nadřazenému objektu.
výchozí:Vector2( 0, 0 ) globalPosition globalPosition

Globální 2D pozice tohoto objektu.


K privátnímu fieldu, se přistupuje pomocí vlasností, které jsou veřejné, pro objekt.

Vlastnost set zajišťuje, automatické zavolání funkce, pro výpočet globální pozice.

Konstruktor inicializuje vlastnosti objektů své třídy a nemá žádné parametry.

Aktualizuje globální pozici, pomocí lokální pozice, nadřazeného objektu.

Pokud má objekt své vlastní podřízené objekty, tak jim aktualizuje globální pozici. Pokud nemá, tak se aktualizuje globální pozice, svého objektu.


Tuto funkci je dobré vyvolat tehdy, když je potřeba aktualizovat globální pozice objektů ve své vlastní scéně.


Příkladem je, pokud chceme změnit velikost renderingu, pomocí resize eventu.


this.resizeHandler = () => { this.renderer.setSize( window.innerWidth, window.innerHeight ); this.pieChart.position = this.globalPosition; this.scene.updateWord(); }
