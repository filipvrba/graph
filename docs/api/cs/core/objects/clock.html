Clock

Objek pro sledování času.

Popis

Tento objekt sleduje čas a je vyvolán ve smyčce, která se spouští na snímkové frekvenci vašeho CPU.

Příklad

/** * Funkce render je vyvolána v loopu a vyšle signál update. * Hodnota delta time, je předána do objektu s názvem dt. */ render( scene ) { scene.emitSignal( { type: 'update', dt: clock.getDT() } ); }

Konstruktor inicializuje privátní vlastnost objektu své třídy a nemá žádné parametry.

Získá uplynulé milisekundy od doby, kdy byla vyvolána funkce getDT() a nastaví na aktuální čas.


vrací časový úsek delta time;
