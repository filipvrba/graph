Animation

Obsahuje informace o vlastnostech objektů, které budou přečteny v přehrávači animací.

Popis

Animace jsou rozděleny do stop, které musí být propojeny s vlastností objektu. Stav vlastnosti, lze přidávat do časových klíčů, a jsou ukládány tak, že jim přidělíme čas a hodnotu k dané vlastnosti.



Animace jsou pouze datové kontejnery, které se přidávají do objektu, jako je AnimationPlayer, aby se mohly přehrát.

Příklad

// Vytváříme animaci, která zvětšuje šířku objektu. const animation = new Animation(); const trackID = animation.addTrack( 'width' ); animation.addInsertKey(trackID, 0, 0); animation.addInsertKey(trackID, 3, 10); animation.addInsertKey(trackID, 7, 20); výchozí:Map()

Veškeré stopy pro animaci, najdeme v této vlastnosti objektu.

Konstruktor inicializuje vlastnost objektu a nemá žádné parametry.
