Popis

Konstanty jsou identifikátory, pro organizaci zdrojového kódu ve třídách.


Místo používání čísel, se používá pojmenovaná konstanta, pro lepší orientaci.

výchozí:'Scene'

Název třídní scény.

výchozí:0

Hodnota dimenze pro valuesMask vlastnost, která je vyvolána v procesu animace.

výchozí:1

Hodnota dimenze pro valuesMask vlastnost, která je vyvolána v průběhu dokončení animace.

výchozí:2

Hodnota dimenze pro valuesMask vlastnost, která je vyvolána pro kontrolu animace.
