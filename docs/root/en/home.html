capitalized( DOCUMENT ) definition

An experimental project called capitalized( DOCUMENT ) uses json data to illustrate embedded data. Using ready-made elements that may be added to your web sites, graphs are visualized.


I created a PieChart graph using an element that shows the typical download speed in Mbps as an illustration.



Using the buttons on the PieChart graphic's bottom, you may make dynamic changes to the interactive chart.

Engine

A core engine written in vanilla Java-script and developed by AUTHOR powers the graph.



The base of Java-script is referred to as Vanilla. Modules from outside sources are thus not implemented in projects with this categorization.
